# Drought Propagation Analysis

This repository contains Python scripts developed during my MSc research at the University of Moratuwa.

## Features

- SPI (1, 3, 12 month)
- SSI (1, 3, 12 month)
- Gamma distribution fitting
- Pearson correlation analysis
- Correlation heatmap visualization

## Libraries

- pandas
- numpy
- scipy
- matplotlib
- seaborn
